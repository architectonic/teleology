# teleology

`teleology` is deprecated as a standalone package name.

Use:

```bash
npx architectonic add doctrine
```

Compatibility alias:

```bash
npx architectonic add teleology
```

## Reason

Telos is one doctrine component, not the whole governing package.

The canonical package is now:

```text
architectonic/doctrine
```

`doctrine` owns purpose, ethics, epistemology, ontology, governance, oikonomia, methodology, and related runtime-neutral principles.

This repository remains only as a temporary compatibility pointer until it is deleted or archived.
